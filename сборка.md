План для Mac
Шаг 1 — установи Go на Mac:
bash# Через Homebrew (самый простой способ)
brew install go

# Проверить
go version
# go version go1.22.x darwin/amd64
Если Homebrew нет:
bash/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
Шаг 2 — распакуй архив aistack который мы создали:
bashcd ~/Downloads
unzip aistack-mvp-v2.zip
cd aistack
Шаг 3 — замени your-org:
bashfind . -type f \( -name "*.go" -o -name "*.yml" -o -name "*.mod" -o -name "*.sh" -o -name "Dockerfile" -o -name "*.md" \) \
  -exec sed -i '' 's/your-org/workhubonline-soft/g' {} +

Важно: на Mac команда sed требует '' после -i — это отличие от Linux

Шаг 4 — скачай зависимости и проверь сборку:
bashcd cli
go mod tidy          # скачает все пакеты, создаст go.sum
go build ./...       # компиляция — должно пройти без ошибок
go test ./...        # тесты
cd ..
Шаг 5 — скомпилируй бинарник для Linux (именно для сервера, не для Mac):
bashcd cli
GOOS=linux GOARCH=amd64 CGO_ENABLED=0 go build \
  -ldflags="-s -w -X 'github.com/workhubonline-soft/aistack/cmd.version=v0.1.0'" \
  -o ../dist/aistack-linux-amd64 \
  .
cd ..
ls -lh dist/
# aistack-linux-amd64  ~8-12MB
